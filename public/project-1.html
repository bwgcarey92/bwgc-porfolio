Project Overview

Quizzical is a modern web application developed using React and Vite, aimed at offering users an immersive quiz experience. Utilizing the Open Trivia Database API, it dynamically fetches and presents a variety of quiz questions across different categories. Users can start quizzes, select answers with real-time feedback, track their scores, and enjoy a seamless, responsive interface designed for both entertainment and learning purposes.
